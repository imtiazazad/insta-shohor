Question: Local storagevs session storage?

Local storage could keep data even after relading the page where session storage delete data after eloading. Local storage is a parmanant storage where session storage is temorapray. Local storage data should delete manually, on the other hand session storage delete data automatically after reload the session, basically the session expire after having reload. Local storage data doesn't expire but session data expire afte page session end. Local storage stay fixed, no new data is created after opening, on the other hand a new session is created each time a tab or window is opened
